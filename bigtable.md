# BigTable

#### Commands

###### Create Instance

```
$ gcloud beta bigtable instances create spikey-anshul-dev-instance \
    --cluster=spikey-anshul \
    --cluster-zone=us-central1-b \
    --display-name=spikey-anshul-dev \
    --cluster-storage-type=SSD \
    --instance-type=DEVELOPMENT
```
